# 简明步骤

- 下载 node.js
- 安装npm
- 下载Hexo
- 创建github新的仓库
- 配置ssh
- 配置Hexo配置文件 记得repo:地址是SSH 因为gtihub仓库有两个地址一个https 一个 git 一定要用git那个 那样才能用到ssh的自动上传
- 上传测试
